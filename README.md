# MyBotBox Studio SDKs

Official SDKs for executing [MyBotBox](https://mybotbox.com) workflows programmatically.

| Language | Package | Install | Source |
|----------|---------|---------|--------|
| TypeScript | [`@yarlisai/studio-sdk`](https://www.npmjs.com/package/@yarlisai/studio-sdk) | `npm i @yarlisai/studio-sdk` | [`packages/ts-sdk`](./packages/ts-sdk) |
| Python | [`yarlis-studio-sdk`](https://pypi.org/project/yarlis-studio-sdk/) | `pip install yarlis-studio-sdk` | [`packages/python-sdk`](./packages/python-sdk) |

Full documentation: **<https://docs.mybotbox.com>**
