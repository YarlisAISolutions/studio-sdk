# Contributing

This repository is an **automatically generated, read-only mirror** of the TypeScript and
Python SDKs. Its contents are overwritten on every upstream change, so pull requests opened
here cannot be merged.

- **Found a bug?** Open an [issue](https://github.com/YarlisAISolutions/studio-sdk/issues) —
  these are triaged by the MyBotBox team.
- **Want to propose a code change?** The SDK sources live in the MyBotBox platform monorepo
  (`packages/ts-sdk`, `packages/python-sdk`). Reach the team at <support@mybotbox.com> and we
  will route your change through the upstream review + release pipeline.

Thanks for helping improve the MyBotBox SDKs!
